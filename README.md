# E-Commerce Sales & Customer Intelligence Dashboard

## Project Overview

This project is an interactive Business Intelligence dashboard built using Power BI, Power Query, and DAX.

The dashboard analyzes e-commerce sales performance, profitability, customer behavior, regional performance, and product trends to support business decision-making.

The project demonstrates real-world data analytics workflow including data transformation, KPI engineering, dashboard design, drill-down analysis, and interactive reporting.

---

# Objectives

The main objectives of this project were:

* Analyze sales and profitability trends
* Identify top-performing products and categories
* Compare regional business performance
* Build interactive executive-level reporting
* Transform raw data into actionable business insights
